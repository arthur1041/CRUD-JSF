No projeto foi implementado um CRUD utilizando JSF, JPA, MySql, Hibernate e Bootstrap 4 no seu front end.
Os itens implementados foram:
A. Criar uma aplicação java web utilizando JavaServer Faces (JSF)
B. Utilizar persistência em um banco de dados (qualquer banco desejado)
F. Utilizar Bootstrap 4

Para a execução do programa é necessário ter o JVM e JDK instalados em sua máquina e utilizar o arquivo .sql que está na raiz do projeto para fazer a criação do banco.
